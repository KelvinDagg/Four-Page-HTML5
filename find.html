2. SEO Best Practices

A summary of the techniques I learnt in findability


  • Title Tag - Ensure your title reflects the main purpose of the website e.g. Woofs - Dog grooming service. Length, 10 to 70 characters.
  • Meta Description Tag - This states what the purpose of the website is in more detail e.g. Dog grooming service that offers a mobile option that comes to you and pampers your favourite pooch while making their coat silky smooth to the touch. Length 70 to 320 characters.
  • Header Tags - Header tags tell search engines what topics are important. These also prioritise the keyword search algorithms that search engines use.
  • Keyword Consistency - Search engines cross reference keywords to the headings. Keywords should appear consistently in the titles, description and the headers for best results.
  • Backlinks - Having other sites reference back to your website give search engines a strong indication as to your sites authority, relevance and ranking position.
  • Robots.txt - The Robots.txt file lets search engines know what is appropriate for them to crawl through.
  • Sitemap.XML - The sitemap file instructs the search engines how the website is laid out, letting them know the logical order of the content.
  • Usability - A strong multi device and user interaction experience will increase your search engine ranking.
  • Performance - Having a responsive, fast loading web site will ensure the user experience will be more enjoyable.
  • Social Media - Having an active social media presence will instruct search engines that your site is more reputable due to the interconnections.
  • Security - Having an SSL certificate lets search engines know that communications and transactions are more secure.

5. Glossary of Terms

Below is the glossary of SEO terms. They are in order of SEO grouping importance.



Term Meaning
SEO Search Engine Optimisation, is the methods best used to make your website 'findable' to search engines.
Title Tag Brief description of the contents of the web page. Between 10 and 70 characters.
Meta Description Tag More detailed description of what the user can expect to find on your website. Between 70 and 320 characters.
Header Tags Header tags tell search engines important sections within your website; this will also let the search engine know what content belongs to which section.
Content Amount Search engines rank sites based on content. A decent mix of text and other objects improves the ranking.
Image Alt Attributes Image Alt attributes are beneficial for screen reading software and also lets search engines know what the image is about. Always make sure you describe the image as you would to someone over the telephone.
Backlinks Search engines use backlinks to sense the authority of the page content. They will improve your ranking according to how much activity there is pointing to your website.
Broken Links The presence of links that lead to nowhere will affect your sites findability ranking. Broken links are usually '404 - Page not found'
Friendly URLs This refers to how understandable search engines find your web link name. e.g. WEBJUNKIES.INFO search engines do not like the term 'JUNKIES' or the .INFO. They prefer, 'PETSUPPLIES.COM.AU'.
Robots.txt The robots file tells search engines where to look for relevant information about your website. You can also tell the search engines where not to look too; just in case you have folders with private stuff in them
XML sitemaps XML sitemaps tell search engines the structure and grouping of your website. It is like the table of contents of a book, including chapters and location..
Usability Usability is how well your website is visible across devices. This includes load times, aesthetics on mobile devices, and contrasts.
Page speed This is very important. Slow load times and operating times (scripts) will make search engines avoid your website. Good server response times should not exceed 0.6 seconds. All page content loaded should not exceed 4 seconds. All page scripts completed should not exceed 8 seconds.
Page size Large memory useage will move you down the rankings as memory hogs cause high latency, bad for people on mobile devices. A good page size is less than 4MB
GZIP compression GZIP compresses your website prior to sending to the visitor. This saves bandwidth and subsequently improves loading speeds. GZIP is enabled on the webserver side, e.g. Apache web servers
Optimised Images Optimised images assists in decreasing the load time of a web page.
Minification Minification is used on CSS and Javascript files. You can tell if a file is minified if it has .min in the extension. Minification is the removal of carriage returns in files, this increases the web servers processing times.
Social integration This is the presence of social media links usually displayed in the footer of the website. They usually link your website to your Facebook, Twitter, Google+, Instagram, YouTube, LinkedIn, Pinterest and Stumbleupon.
